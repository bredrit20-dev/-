 ## Создание проекта
 Весь проект создавался через **функции** для удобного использования и повтора действий
 
 код первой функции:
 ```Python
 def kal(a, b, c):
    
    prog(a ,b,c)
 ```
 Эта фунция нужна для того, чтобы пользователь мог **начинать** всё с самого начала, он вызывает другую функцию

 код второй функции:
 ```Python
 def prog(a, b, c):
    if c == 1:
        print(a+b)
    
    elif c == 2:
        print(a*b)

    elif c==0:
        kal(a = int(input("Введите 1 число")), b = int(input("Введите 2 число")) , c = int(input("Введите 1 если хотите их сложить, 2 если хотите перемножить и 0 если ввести числа заново")))
 ```
 Эта функция выполняют **основную** задачу кода и считает наши числа, а в том случае если мы хотим ввести другие значения заново выхывает первую функцию которая хранит их

 ```Python
 kal(a = int(input("Введите 1 число")), b = int(input("Введите 2 число")) , c = int(input("Введите 1 если хотите их сложить, 2 если хотите перемножить и 0 если ввести числа заново")))
 ```
 Ну и, наконец, код вызова функции, здесь мы даем пользователю самому **выбрать** значения для перемененых, а также **выбрать** действие с ними

## Создане более простого кода

Итак, мы создали простой калькулятор, но его код достаточно запутанный и не полный

Можно создать **новый**, законченный и более красивый код:
```Python
def prog(a, b, c):
    if c == 1:
        print(a+b)
    
    elif c == 2:
        print(a*b)

    elif c==0:
        prog(a = int(input("Введите 1 число")), b = int(input("Введите 2 число")) , c = int(input("Введите 1 если хотите их сложить, 2 если хотите перемножить и 0 если ввести числа заново")))
    else:
        print("Неправильное число")
        prog(a = int(input("Введите 1 число")), b = int(input("Введите 2 число")) , c = int(input("Введите 1 если хотите их сложить, 2 если хотите перемножить и 0 если ввести числа заново")))
    
prog(a = int(input("Введите 1 число")), b = int(input("Введите 2 число")) , c = int(input("Введите 1 если хотите их сложить, 2 если хотите перемножить и 0 если ввести числа заново")))

```

Этот код имеет всего одну функцию, а также начинается заново если пользователь введет неправильное значения, он значительно **проще** и **практичнее** чем предыдущий.
